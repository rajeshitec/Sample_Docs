# User workflows

This page summarizes common end-to-end workflows.

## Create a new client

1. Go to **Clients → New client**.
2. Enter mandatory KYC fields.
3. Submit for approval.
4. Monitor status under **Clients → Pending**.

## Capture a trade

1. Go to **Trades → New trade**.
2. Select client, product, and trade terms.
3. Validate and submit.
4. Track lifecycle in **Trades → Activity**.
