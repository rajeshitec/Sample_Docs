# Getting started

This section helps new users become productive with **Project** quickly.

## Access prerequisites

1. Corporate network or VPN access  
2. Active account in the Identity Management system  
3. Supported browser (Chrome / Edge / Firefox, latest 2 versions)

If you don't have access yet, follow [Onboarding](onboarding.md).

## First login

1. Go to `https://project.internal.corp`
2. Sign in with corporate SSO.
3. Complete profile confirmation and MFA setup if prompted.

If you see *Access denied*, contact support with your username and department.

## Main navigation

- **Dashboard** – KPIs, recent activity, alerts  
- **Clients** – client search and maintenance  
- **Trades** – trade capture and lifecycle  
- **Reports** – exports for ops / compliance  
- **Admin** – configuration and user management (admins only)

See [User workflows](user-workflows.md) for end-to-end flows.
